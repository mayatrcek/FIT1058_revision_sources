## Part A — Multiple choice (single answer)

**Q1.** To prove the statement "if n² is even then n is even," which method is most natural and why?
- (a) Direct proof — assume n² even and factor it
- (b) Contrapositive — prove "if n is odd then n² is odd"
- (c) Proof by cases on the parity of n²
- (d) Counterexample

**Q2.** A proof by contradiction of "√2 is irrational" begins by assuming:
- (a) √2 is irrational and deriving a contradiction
- (b) √2 = a/b in lowest terms (a, b integers, not both even) and deriving a contradiction
- (c) 2 = a²/b² for some integers a, b
- (d) √2 is rational and that a/b is already in lowest terms is irrelevant

**Q3.** Which of the following, on its own, is a *valid* proof that "for all integers n, n² + n is even"?
- (a) "It's true for n = 1, 2, 3, so it holds for all n."
- (b) "n² + n = n(n+1); one of two consecutive integers is even, so the product is even."
- (c) "Even numbers are closed under addition."
- (d) "n² is even or odd, and adding n fixes it."

**Q4.** The contrapositive of "if x + y is irrational then x is irrational or y is irrational" is:
- (a) If x is rational and y is rational, then x + y is rational
- (b) If x is irrational or y is irrational, then x + y is irrational
- (c) If x + y is rational, then x is rational and y is rational
- (d) If x is rational or y is rational, then x + y is rational
