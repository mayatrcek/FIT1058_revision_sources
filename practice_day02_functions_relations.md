## Section A — Multiple choice (single answer)

**Q1.** Let f : ℝ → ℝ be defined by f(x) = 3x − 7. Which statement is true?
- (a) f is injective but not surjective
- (b) f is surjective but not injective
- (c) f is bijective
- (d) f is neither injective nor surjective

**Q2.** How many functions are there from a set A with |A| = 4 to a set B with |B| = 3?
- (a) 3⁴ = 81
- (b) 4³ = 64
- (c) 4·3·2·1 = 24
- (d) 3·2·1 = 6

**Q3.** Let A = {1, 2, 3}. Which of these relations on A is an equivalence relation?
- (a) {(1,1),(2,2),(3,3),(1,2)}
- (b) {(1,1),(2,2),(3,3),(1,2),(2,1)}
- (c) {(1,1),(2,2),(3,3),(1,2),(2,3),(1,3)}
- (d) {(1,2),(2,1),(1,1),(2,2)}

**Q4.** If f : A → B and g : B → C are both injective, then g ∘ f is:
- (a) necessarily injective
- (b) necessarily surjective
- (c) necessarily bijective
- (d) not necessarily any of these

## Section B — Select all that apply

**Q5.** Let R be the relation on ℤ defined by a R b ⇔ a − b is divisible by 5. Select ALL properties R has.
- (a) Reflexive
- (b) Symmetric
- (c) Antisymmetric
- (d) Transitive

**Q6.** Which of the following functions f : ℝ → ℝ are injective? (Select all.)
- (a) f(x) = x²
- (b) f(x) = x³
- (c) f(x) = 2x + 1
- (d) f(x) = |x|

State your method, define your variables, and justify each line. (Note: g need not be injective — this is exactly the kind of subtle direction the exam tests.)
